Game Summary

Blockchain is a puzzle game that revolves around chaining various blocks together. Each block has different properties that allows them to be able to stick to some other blocks or slide past them. You must create different shapes in order to avoid falling into the abyss below. Navigate your way to the white button in each level to win!
